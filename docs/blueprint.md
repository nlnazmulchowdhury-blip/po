# **App Name**: CinemaStream

## Core Features:

- Home Page & Categories: YouTube-inspired layout with category-based video grids and featured content banners.
- Smart Search & Filters: Enables users to find movies by name, category, release year, or cast members easily.
- Custom YouTube-like Player: A robust player featuring play/pause, volume control, fullscreen, resolution switching, and playback speed control.
- Social Interaction & Comments: Allows users to like, dislike, and leave comments on movies exactly like the YouTube experience.
- Watchlist & Favorites: Users can bookmark movies to a 'Watch Later' or favorites list for future viewing.
- Admin Analytics Dashboard: A centralized control room providing a graphical overview of user statistics, movie uploads, and daily view counts.
- Content & User Management: Admin tools for movie uploads (server or link embed), metadata editing, user moderation, and comment cleanup.
- Secure Auth & Roles: Authentication system with encrypted passwords and social login, plus role-based access for admins and users.
- Streaming Optimization & CDN: High-performance video delivery using cloud storage like AWS S3 or Cloudinary and Content Delivery Networks.
- Watch History Tracking: Maintains a record of previously viewed content so users can pick up where they left off.
- Dynamic Theme Switching: Toggle between dark and light modes to customize the visual experience based on user preference.

## Style Guidelines:

- Primary color: Vibrant Indigo (#5E5CE6) for a premium feel. Background: Deep Midnight (#0C0C0E) for eye comfort. Accent: Bright Azure (#0A84FF) for progress bars.
- Headlines use 'Space Grotesk' for a modern look, while body text uses 'Inter' for high legibility.
- Outlined, minimal icons following the familiar YouTube design language for cross-device consistency.
- Mobile-first responsive grid system with a persistent sidebar for navigation on desktop screens.
- Smooth category transitions and subtle hover-scale effects on movie thumbnails for a tactile feel.
